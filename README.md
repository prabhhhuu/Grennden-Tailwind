# Greenden – Plant Store Website

## Overview

Greenden is a modern and responsive plant store website built using HTML and Tailwind CSS. The project focuses on creating a clean user interface, responsive layouts, and an engaging shopping experience for users interested in indoor and outdoor plants.

The website is designed to showcase plant collections, highlight featured products, and provide an easy way for visitors to explore and purchase plants online.

## Features

- Responsive design for mobile, tablet, and desktop devices
- Modern user interface built with Tailwind CSS
- Interactive product cards with hover effects
- Featured and best-selling plant collections
- Customer review section
- Contact page for customer inquiries
- Smooth transitions and animations
- Clean and organized code structure

## Technologies Used

- HTML5
- Tailwind CSS 2.2.19
- CSS3
- JavaScript (if applicable)

## Project Structure

- `index.html` – Home page
- `product.html` – Product listing page
- `contact.html` – Contact page
- `images/` – Project images and assets
- `tailwind.config.js` – Tailwind CSS configuration
- `postcss.config.js` – PostCSS configuration
